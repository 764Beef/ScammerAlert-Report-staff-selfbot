## 🎟 Ticket Management

- `.lt` — List tickets you are currently handling
- `.ls <userID>` — Find report listings for a specific user
- `.clear` — Remove invalid or closed tickets
- `.re <channelID>` — Remove a specific ticket
- `.unc` — Display tickets with no staff response
- `.inactive <time>` — Show inactive tickets (e.g., `1d`, `2h`)

## 📢 Notifiers

- `.dm <userID>` — Send a direct message regarding an open report
- `.def <userID>` — Remind a user to submit a defense
- `.d <userID>` — Mark a report as denied
- `.a <userID>` — Mark a report as accepted
- `.r <userID>` — Mark a report as resolved

## 🏆 Leaderboard & Reports

- `.lb` — Display leaderboard for the current month
- `.lb <DD MM YYYY>` — Display leaderboard from a specific date
- `.calb` — Recache the last 50 messages to refresh leaderboard data
- `.pc` — Correct miscounted leaderboard entries
- `.cleanlb` — Remove duplicate entries from the leaderboard database

## 📝 Form Templates & Report Guidelines

- `.fr` — Report form template
- `.ex` — Explanation of correct payment format
- `.ids` — Reminder to use correct user IDs
- `.au` — Autobuy report requirements
- `.cr` — Conversation and payment proof guidelines
- `.low` — Minimum disputable amount
- `.cur` — Accepted currencies
- `.en` — English evidence requirement
- `.exid` — Example of correct ID formatting
- `.holds` — Mark report as on hold for senior review
- `.holdm` — Mark report as on hold for admin review

## 🛠 Utilities

- `.tm <time>` — Set a timer (e.g., `.tm 10m`, `.tm 2h`)
- `.help` — Display command help menu
