# Configuration Guide

This project requires a Discord account token to operate.

---

## 🔐 Token Setup

Create or locate the file:

```
token.txt
```

Place your Discord account token inside the file


The file should contain **only the raw token string** and nothing else.

---

## 📁 File Location

Ensure `token.txt` is located in the root directory of the project:

```
ScammerAlert-Repost-Staff/
│
├── token.txt
├── package.json
└── ...
```
---

After adding your token, you can start the application:

```bash
npm start
```
